# Final Submission Guide

## Before GitHub
- Run `npm install`
- Run `npm test`
- Run `npm run build` after adding env vars
- Replace TODOs in README, DEVLOG, REFLECTION, USER_INTERVIEWS

## Required real parts
Do not fabricate these:
- 3 user interviews
- 7 devlog entries
- git commits across distinct days
- screenshots/video
- deployed URL

## Cloudflare Pages
Connect GitHub repo. Build command: `npm run build`. Add all env variables from `.env.example`.

## Supabase table
Create a `leads` table with email, company, role, team_size, audit jsonb, created_at.

## Final form
Submit public GitHub URL + live deployed URL.
